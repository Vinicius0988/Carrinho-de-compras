# 🛒 Carrinho de Compras com React JS

Este é um projeto de **carrinho de compras** desenvolvido com **React JS**, que simula um e-commerce utilizando uma **Fake API** para listar, buscar e manipular produtos.

---

## ✨ Funcionalidades

- 🔍 **Busca de produtos** — Pesquise produtos por nome utilizando uma API falsa.
- 🛍️ **Listagem de produtos** — Visualize os produtos em cards com imagem, nome e preço.
- ➕ **Adicionar ao carrinho** — Inclua produtos no carrinho com controle de quantidade.
- ➖ **Remover produtos** — Remova uma unidade ou todos os itens de um produto.
- 💰 **Resumo do carrinho** — Veja o valor total dos itens adicionados em tempo real.

---

## 🛠️ Tecnologias Utilizadas

- ⚛️ **React JS** — Biblioteca JavaScript para construção de interfaces.
- 💅 **CSS3** — Estilização moderna e responsiva.
- 📦 **HTML5** — Estruturação da aplicação.
- 🌐 **Fake Store API** — Simula dados reais de produtos sem necessidade de autenticação.
